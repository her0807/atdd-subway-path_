# 2단계 - 요금 정책 추가
## 기능 요구 사항 목록

- [x] 노선별 추가 요금
  - [x] 추가 요금이 있는 노선을 이용할 경우 측정된 요금에 추가
- [x] 연령별 요금 할인
  - [x] 아이, 노인: 무료
  - [x] 청소년: 운임에서 350원을 공제한 금액의 20%할인 
  - [x] 어린이: 운임에서 350원을 공제한 금액의 50%할인
  - [x] 일반 : 할인되지 않음
